# Портфолио по анализу данных в Excel

В этом репозитории собраны три практических кейса, которые я выполнила для зачета. Проекты включают в себя разные задачи: от базовой сортировки до подготовки данных для тестов и анализа продаж крупного бизнеса.

## Что внутри:

1. **`Assignment.xlsx`**
   * **Diagrams**: построила наглядные диаграммы, которые показывают динамику продаж и доли основных продуктов.
   * **Structuring**: привела массив данных в удобный табличный вид и сгруппировала интернет-трафик по интервалам.
   * **AB test**: разделила базу клиентов на две одинаковые группы (тестовую и контрольную) для проведения A/B тестирования.

2. **`Excel test case Novartis Pharma Services AG .xlsx`**
   *  Я рассчитала общие начисления абонентам компании связи, продолжительность разговоров, направления звонков, среднюю продолжительность разговоров и т.д.

3. **`Тестирование Аналитик .xlsx` (Аналитические задачи)**
   * Практическое задание, где нужно было объединить две разные таблицы, построить топ-3 территорий по разным показателям, а также построить график с понедельной динамикой товарооборота и товарооборота на склад в целом по всем территориям.

## 🛠️ Какие навыки я продемонстрировала:
* **Работа с формулами**: использование связок функций (ВПР/XLOOKUP, СУММЕСЛИМН, ЕСЛИ и др.) для расчетов.
* **Инструменты Excel**: создание и настройка сводных таблиц (Pivot Tables), фильтрация и группировка данных.
* **Визуализация**: построение понятных графиков для презентации результатов.
* **Подготовка данных**: очистка «грязных» таблиц и формирование выборок для анализа.
